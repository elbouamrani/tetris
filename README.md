# tetris

![tetris demo](https://github.com/elbouamrani/tetris/blob/master/public/tetris-demo.gif)

## controls

| key         | action     |
| ----------- | ---------- |
| arrow up    | rotate     |
| arrow right | move right |
| arrow left  | move left  |
| arrow down  | accelerate |
| space bar   | play/pause |
